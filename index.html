本批可走查

REQ-E2E-001
管理員登入並檢視總覽
  1. 進入總覽頁（本批原型直接進入；登入頁於下一批補上）
  2. 頂部 KPI 卡列：確認 6 張卡、各級距數字「同時攤開」不需下拉（協理要求）
  3. 滑過任一指標旁的 ⓘ 小圖示 → 顯示計算口徑說明
  4. 「承接任務的分布」：切換縣市／月份 → 圖表即時變化；滑過柱條看數字
  5. 「隊員承接數」排卡：切換週／月、移除卡片、點「＋新增指標卡」從指標池組一張新卡
  6. 「進行中活動」：確認成效（+任務數）與費用（成本）是視覺重點；級距前後對比條
  7. 「早晚尖峰完成數」：切換年份 → 圖與表更新
  8. 右下角「原型狀態」切換：空狀態／載入中／錯誤／權限不足 各看一眼
開始走查 → 總覽
REQ-OV-007（總覽範圍・其他分頁）
早晚尖峰完成數（對應現行 Tableau 三張報表）
  1. 上方三個 tab 依序看：「完成數 KPI」（年選擇＋早晚雙柱圖＋月表）
  2. 「每日完成數 by 星期」：同星期別日期併列，早／晚尖峰各含有效任務數、完成數、完成率、日均完成（累計）
  3. 「每日任務數・完成狀況」：每日完成／取消／未現／有車無承接／搜無車 數量與 %，可切早／晚尖峰
  4. 右上切換地區（雙北／全台）、年、月 → 三張報表同步更新；「匯出」為示意
開始走查 → 尖峰完成數
REQ-E2E-006（部分）＋活動→Profile 鏈路
隊員 Profile（協理第一階段最重要）＋主管確認路徑
  1. 總覽 →「隊員活動」任一活動卡 → 點「查看參加隊員」→ 名單彈窗 → 點任一隊編 → 進入該隊員 Profile
  2. Profile 頁：確認 11 維度資料卡（上線天數、早晚尖峰、身份別、承接率、活動參與、月均承接、收入、常跑區域、不願承接原因）
  3. 滑過各維度 ⓘ 看口徑；「收入」「電話」有敏感欄位標示（原型以已授權視角展示）
  4. 「趨勢」切換 承接任務數／收入 × 週／月／季
  5. 下方「溝通歷程」時間軸：每次聯繫的時間、專案、內容、意願（對應協理：連動看到何時溝通過、溝通什麼方案）
  6. 搜尋列直接輸入隊編（示例 chips：123456／0103／0217）；輸入不存在的隊編看「查無此隊編」
開始走查 → 隊員 Profile
